# 行业风格提示词模板

根据行业类型从以下模板中随机抽取「风格数量」种风格。

| 风格名 | 提示词 |

| 北境枭雄 | 厚重黑色毛领金属铠甲，冷峻蓝调光线，史诗级电影感 |
| 云海剑仙 | 月白色宽袖道袍，水墨山水暗纹，儒雅仙侠气质 |
| 幽冥夜行 | 哑光黑色贴身夜行衣，神秘暗影刺客 |
| 黄金铠甲 | 华丽黄金铠甲，猩红丝绸斗篷 |

```sql
-- 示例：游戏行业选 2 种风格
INSERT INTO video_style VALUES
('北境枭雄', '厚重黑色毛领金属铠甲，冷峻蓝调光线，史诗级电影感'),
('云海剑仙', '月白色宽袖道袍，水墨山水暗纹，儒雅仙侠气质')
ON CONFLICT (name) DO UPDATE SET prompt = EXCLUDED.prompt;
```

## 使用方式

1. 根据用户提供的 **行业类型** 选择对应模板库
2. 根据用户要求的 **风格数量**（默认1，最多4）随机抽取
3. 用抽取的风格名和提示词填入 `INSERT INTO video_style` 语句
4. 可根据用户的 **视觉风格** 描述微调提示词
